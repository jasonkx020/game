# 房卡场运营 SOP

> 俱乐部孵化与房卡复购标准流程。概览见 [overview.md](overview.md)。

---

## 1. 俱乐部孵化流程

【运营动作】按以下节奏推进新俱乐部：

### Day 1~3：建群

- 代理协助管理员创建微信群/QQ 群（30~50 人）
- 发放 5 张体验房卡给每位新成员

### Day 4~7：首局

- 管理员组织首场 4 人局
- 【运营动作】客服在线支持，确保首局体验流畅

### Week 2~4：稳定组局

- 目标：每周组局 ≥ 2 次
- 【运营动作】每周推送组局提醒给俱乐部管理员

### Month 2+：代充模式

- 管理员批量采购房卡，成员无感消费
- 平台给管理员 3~5% 流水返点（见 [agent-club.md](agent-club.md)）

---

## 2. 房卡复购促进

【运营动作】

| 触发条件 | 动作 |
| :--- | :--- |
| 房卡余量 < 5 张 | 系统推送补货提醒 |
| 连续 7 日未组局 | 推送「老友局」活动邀请 |
| 俱乐部月 GMV 达标 | 推送阶梯返利进度 |

---

## 3. 地推日常 checklist

| 频率 | 事项 |
| :--- | :--- |
| 每日 | 跟进新俱乐部首局进度 |
| 每周 | 统计俱乐部组局频率，低于 2 次/周的重点跟进 |
| 每月 | 代理 KPI Review、阶梯返利结算 |

---

## 4. 合规要求

对外推广话术遵循 [risk/compliance.md](../../risk/compliance.md)，强调「休闲娱乐、亲友组局」。

代理风控（保证金、实名）遵循 [risk/anti-fraud.md](../../risk/anti-fraud.md)。
