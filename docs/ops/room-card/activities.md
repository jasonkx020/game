# 房卡场专属活动

> 仅适用于房卡场 / 地推渠道。记分场活动见 [score-field/activities.md](../score-field/activities.md)。

---

## 1. 周期性活动

| 周期 | 活动 | 内容 | 目的 |
| :--- | :--- | :--- | :--- |
| **每日** | 任务 | 玩 3 局领 1 房卡碎片，5 碎片合成 1 房卡 | 日活 + 房卡消耗 |
| **每周** | 俱乐部积分榜 | 俱乐部内积分排名，周冠军奖 5 房卡 | 俱乐部粘性 |
| **每周** | MVP 免房卡 | 本周 MVP 免 1 局房卡 | 激励活跃 |
| **每月** | 代理阶梯返利 | 进货 500 张返 5%，1000 张返 8% | 渠道动力 |

---

## 2. 俱乐部积分榜

| 项目 | 规则 |
| :--- | :--- |
| 排名依据 | 俱乐部内成员累计积分（按游戏规则结算） |
| 周期 | 每周一 0:00 重置 |
| 奖励 | 周冠军俱乐部管理员奖 5 房卡 |
| 公示 | 俱乐部后台 + 微信群推送 Top 10 |

---

## 3. MVP 免房卡

| 项目 | 规则 |
| :--- | :--- |
| 评选依据 | 本周局数最多 + 胜率综合排名 |
| 奖励 | 免 1 局房卡消耗 |
| 范围 | 单个俱乐部内 |

---

## 4. 代理阶梯返利

| 月进货量 | 返利比例 |
| :--- | :--- |
| 100~499 张 | 0% |
| 500~999 张 | 5% |
| 1,000 张+ | 8% |

详见 [agent-club.md](agent-club.md) §7。

---
