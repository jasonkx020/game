# 反作弊与代理风控

> 风控层 — 横切所有场域与游戏。合规话术见 [compliance.md](compliance.md)。

---

## 1. 同桌风控

| 检测项 | 阈值 | 动作 |
| :--- | :--- | :--- |
| 同 IP | 最多 3 人同桌 | 第 4 人拒绝入座 |
| GPS 距离 | 200m 内告警 | 标记可疑，人工复核 |
| 同设备 | 最多 2 账号 | 第 3 账号拒绝登录 |
| 异常输赢 | 24h 内赢/输超过均值 5 倍 | 人工复核，必要时冻结 |

【产品需求】风控面板展示：同桌 IP/GPS 分布、异常输赢列表、刷子检测结果。

---

## 2. 异常输赢复核流程

```
系统检测异常 → 自动标记 → 拉取该局 replay（round_id）→ 24h 内人工复核 → 判定 → 通知用户
```

【运营动作】配备专职风控客服，工作日 24h 内完成复核。

### 2.1 技术依据

| 工具 | 用途 |
| :--- | :--- |
| `GET /v1/rounds/{round_id}/replay` | 完整局回放（含出牌时序） |
| `GET /v1/admin/audit/actions?audit_sn=` | 精确定位某步操作及前后 context |
| `game_action_log.action_seq` | 验证是否存在异常协作出牌模式 |

异常标记时应记录关联 `round_id` 与 `settlement_audit_sn`，便于一键跳转回放。

## 3. 代理风控

| 规则 | 说明 |
| :--- | :--- |
| 实名认证 | 一级代理必须实名 |
| 保证金 | 5,000 元，违规扣罚 |
| 扣罚情形 | 虚假推广、违规宣传、用户投诉、刷单 |
| 提现审核 | 返点提现 T+7，人工审核 |

详见 [ops/room-card/agent-club.md](../ops/room-card/agent-club.md)。

---

## 4. 场域适用说明

| 规则 | 房卡场 | 记分场 |
| :--- | :--- | :--- |
| 同 IP / 设备限制 | 适用 | 适用 |
| 异常输赢复核 | 适用（积分/房卡异常） | 适用（金币异常） |
| 代理保证金 | 适用 | 不适用 |

运营 SOP 不重复本模块阈值，仅链接引用。
