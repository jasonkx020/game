# 增值运营（Phase 3）

> 跨场域 VIP、赛事、道具。Phase 3 上线，6 月+。  
> 涉及**游戏金币**的权益以当前游戏 ops-hooks 为准（示例：打乌龟 [dawugui/ops-hooks.md](../../games/dawugui/ops-hooks.md)）。

---

## 1. 上线节奏

**时间：** Phase 2 记分场完成后 6 月+  
**目标：** 利润优化，提升 ARPU 与 LTV

---

## 2. 产品定价

| 模块 | 定价 | 内容 | 预期贡献 |
| :--- | :--- | :--- | :--- |
| VIP 月卡 | 18 元/月 | 每日领游戏金币（数量见 ops-hooks）、专属头像框、战绩详析 | 稳定订阅收入 |
| 赛季通行证 | 30 元/季 | 任务驱动日活，分级奖励 | 提升 7 日留存 |
| 周末锦标赛 | 报名费 = 底分 × 20 | 以**该游戏金币**支付，平台抽 10% | 高净值用户 |
| 装扮商城 | 6~68 元 | 牌桌皮肤、出牌特效 | 冲动消费 |
| 记牌器 | 1~6 元 | 单局/单日 | 纯利润 |
| 战绩导出 | 3 元/次 | PDF 报告 | 俱乐部需求 |

---

## 3. VIP 权益明细

签到等金币奖励按**当前游戏**独立配置：

| 权益 | 普通用户 | VIP 用户 |
| :--- | :--- | :--- |
| 每日签到 | 见 ops-hooks | 见 ops-hooks（通常 2 倍） |
| 头像框 | 默认 | 专属 VIP 框 |
| 战绩详析 | 基础 | 完整（含游戏特殊状态统计） |
| 客服响应 | 标准 | 优先 |
| 破产救济 | 1 次/日 | 3 次/日 |

**打乌龟示例：** 签到 200 / 400 龟币 — 见 [dawugui/ops-hooks.md](../../games/dawugui/ops-hooks.md) §2.3。

---

## 4. 周末锦标赛规则

| 项目 | 规则 |
| :--- | :--- |
| 时间 | 每周六 20:00 ~ 22:00 |
| 报名费 | 以**该游戏金币**支付，底分 × 20（底分见 ops-hooks） |
| 平台抽成 | 报名费的 10% |
| 奖励分配 | 前 3 名分奖池 80%，4~10 名分 20% |
| 赛制 | 积分制，2 小时内心积分为准 |
| 运营主体 | 记分场官方 |

**打乌龟示例：** 初级场 500 × 20 = 10,000 龟币 — 见 [dawugui/ops-hooks.md](../../games/dawugui/ops-hooks.md)。

【运营动作】锦标赛前 3 天推送报名提醒，前 10 名发放「锦标赛冠军」称号。

详见 [score-field/activities.md](../score-field/activities.md)。
