# 合规与宣传规范

> 风控层 — 横切所有场域与游戏。同桌风控见 [anti-fraud.md](anti-fraud.md)。

---

## 1. 对外宣传规范

| 场景 | 合规话术 | 禁止话术 |
| :--- | :--- | :--- |
| 房卡场 | 休闲娱乐、亲友组局 | 赌博、赢钱、提现 |
| 记分场 | 竞技娱乐、虚拟游戏金币 | 真钱、兑换、赌博 |
| 代理推广 | 休闲游戏推广 | 投资、躺赚、赌博 |

【运营动作】所有对外素材（海报、文案、视频）需经合规审核后发布。

---

## 2. 货币规则

### 2.1 房卡（平台通用）

| 规则 | 说明 |
| :--- | :--- |
| 绑定平台 | 全游戏通用，用于开房 |
| 不可局内输赢 | 仅消耗于创建/续费房间 |
| 不可玩家间交易 | 防止场外交易 |

### 2.2 游戏金币（各游戏独立）

| 规则 | 说明 |
| :--- | :--- |
| 绑定游戏 | 各游戏独立币种（如龟币），仅用于该游戏记分场 |
| 不可跨游戏转移 | 打乌龟金币不可用于其他游戏 |
| 不可跨游戏兑换 | 不同游戏金币不可互相转换 |
| 单向充值 | 人民币 → 游戏金币，不可逆 |
| 不可提现 | 游戏金币不可兑换人民币或实物 |
| 不可玩家间交易 | 防止场外赌博 |
| 不可兑换房卡 | 防止套利 |

---

## 3. 未成年人保护

| 规则 | 说明 |
| :--- | :--- |
| 实名认证 | 所有用户必须实名 |
| 防沉迷 | 未成年人 22:00~8:00 禁止游戏 |
| 充值限制 | 未成年人禁止充值 |
| 游戏时长 | 未成年人单日 ≤ 1.5 小时 |
